# baicai.dev

A simple personal site for learning in public, small projects, notes, and contact information.

## Structure

- `/` - homepage
- `/about` - short personal introduction
- `/projects` - project list
- `/notes` - lightweight notes
- `/contact` - contact details

## Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- Vercel

## Local development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
